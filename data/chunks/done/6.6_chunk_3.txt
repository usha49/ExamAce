Translation: 
A point is translated from position P=(x,y,z) to position P’ = (x’,y’,z’) with the matrix operation 
 
      x’              1    0    0    tx             x 
      y’      =     0    1    0    ty      .      y 
      z’              0    0    1    tz             z 
      1               0    0    0    1              1 
 
or     P’ = T.P 
 
Parameters   tx   , ty    ,tz    specify translation distances for the coordinate directions x ,y and z.
